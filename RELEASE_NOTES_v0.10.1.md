# ByteCase Playbooks v0.10.1 Release Notes

## Summary

v0.10.1 removes the repetitive quick navigation bar, renames Scenarios to Coaching, and reworks the Artifacts lane so it starts with the examiner question and situation rather than an artifact list.

## Changes

- Removed the persistent quick navigation bar.
- Renamed the Workbench subtab from Scenarios to Coaching.
- Rebuilt Coaching as a mentoring-examiner prompt flow.
- Added multiple-choice best-response coaching prompts.
- Added coaching explanations, follow-up questions, and bigger-picture takeaways.
- Added starter prompts for powered-on BitLocker systems, command attribution, and order of volatility.
- Reworked Artifacts so the user starts with situation + question.
- Mapped questions to artifact areas, locations, common tools/methods, guardrails, and documentation reminders.
- Added clearer wording that tool names are common examples, not endorsements, recommendations, or validation statements.

## Scope

Playbooks remains a reference and readiness tool. It does not perform forensic analysis, make findings, write case notes, track case progress, or replace formal training, agency policy, legal authority, tool validation, or examiner judgment.
