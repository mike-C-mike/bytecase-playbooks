# ByteCase Playbooks v0.10.0 Release Notes

## Release focus

v0.10.0 is a navigation consolidation release. It pushes Playbooks toward a clearer reference/readiness identity by reducing top-level navigation and grouping related tools into fewer lanes.

## Highlights

- Reduced top-level navigation to Home, Workbench, and Library.
- Moved Guide, Playbook, Artifacts, Scenarios, and Drills into Workbench.
- Moved Question Packs under Library beside Search, Save / Export, and Settings.
- Added a persistent quick navigation bar for common jumps.
- Updated README and added a navigation model document.
- Preserved all existing core functionality.

## Scope reminder

ByteCase Playbooks does not create case notes, track case progress, perform evidence parsing, or make investigative conclusions.

## Recommended validation before release

- Launch GUI and confirm all top-level lanes open.
- Confirm Quick nav buttons route correctly.
- Open a playbook from Home.
- Run a Coach drill from Workbench > Drills.
- Import or validate a sample question pack from Library > Question Packs.
- Export a playbook session from Library > Save / Export.
- Search glossary/playbook/coach content from Library > Search.
