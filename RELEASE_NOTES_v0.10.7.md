# ByteCase Playbooks v0.10.7 - UI Polish Sprint

## Summary

This sprint focuses on visual polish and navigation clarity. No new forensic or coaching capabilities were added.

## Changes

- Increased default window size to 1280x820 and minimum size to 1120x720.
- Shortened Workbench tab labels: Guide, Steps, Artifacts, Coach, Drills.
- Shortened Library tab labels: Export, Search, Packs, Settings.
- Removed repeated long explanatory language from key panels.
- Shortened common button labels for consistency.
- Reduced oversized text boxes where content felt cramped or visually heavy.
- Cleaned panel names across Home, Guide, Artifacts, Coach, Drills, Search, Packs, and Settings.
- Kept Playbooks focused as a reference and readiness module.

## Boundary

ByteCase Playbooks remains a reference and readiness tool. It does not track case progress, create case notes, perform acquisition, parse evidence, or generate investigative findings.
