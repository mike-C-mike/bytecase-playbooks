# ByteCase Playbooks v0.10.2 Release Notes

## Artifact Library Expansion Sprint

This sprint expands the Artifact section and makes it more context-aware.

### Added

- Situation-specific Question dropdown behavior.
- Expanded artifact areas for Windows execution, Windows file access, Windows logons/sessions, Windows event logs, USB/external media, external media file movement, browser activity, cloud sync, remote access/automation, memory/RAM, BitLocker/live encryption context, file system metadata, mobile messages/media, mobile location/media, and mobile extraction limitations.
- More targeted examiner questions by situation.
- Clearer mapping from situation to decision factors, artifact areas, where to look, common tools/methods, guardrails, and documentation reminders.

### Changed

- Mobile Device Review now shows mobile-specific questions instead of broad Windows-oriented questions.
- The Artifacts lane is more question-driven and less artifact-list-driven.

### Boundary

Tool names are common examples only. They are not endorsements, recommendations, validation statements, or a replacement for agency policy and local validation.
